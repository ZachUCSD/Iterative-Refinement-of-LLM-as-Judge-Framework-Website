# Iterative Refinement of LLM-as-Judge Framework – Website Code

This repository contains the source code for the project website associated with the **Iterative Refinement of LLM-as-Judge Framework**.

## Repository Structure

* **index.md** – Main page content for the website, including project description, methodology, results, and discussion.
* **_config.yml** – Configuration file that sets the Jekyll theme and site settings.
* **_layouts/default.html** – Base layout that defines the overall page structure and formatting constraints.

## Visualizations

The interactive visualizations used on the site are implemented as standalone HTML files:

* **judge_flowchart.html** – Interactive pipeline diagram of the LLM-as-Judge evaluation framework.
* **scoring_pipeline.html** – Visualization of the hybrid scoring system and evaluation metrics.
* **results.html** – Interactive visualization showing summary refinement across iterations.

These visualizations are embedded into the main page using `<iframe>` elements.

## Purpose

This repository serves as the **public-facing implementation of the project website**, presenting the methodology, evaluation framework, and experimental results described in the project report.
